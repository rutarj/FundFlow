
# FundFlow - Alternative Investment Reporting Engine

FundFlow is a Python based fund reporting engine using AWS Glue ETL pipelines and Lambda functions, processing NAV calculations and investor statements from S3 into DynamoDB.
## Tech Stack
- Python
- AWS Glue
- Lambda
- S3
- DynamoDB
- CloudFormation
- Spark
## Features
- ETL pipeline for fund data processing
- NAV calculation engine
- CloudFormation IaC deployment
- Multi currency fund performance reporting
