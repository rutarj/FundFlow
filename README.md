# FundFlow —  Investment Reporting Engine

Python based fund reporting engine using AWS Glue ETL pipelines and Lambda functions.
Processes NAV calculations and investor statements from S3 into DynamoDB.

## Tech Stack
Python, AWS Glue, Lambda, S3, DynamoDB, CloudFormation, Spark

## Features
- ETL pipeline for fund data processing
- NAV calculation engine
- CloudFormation IaC deployment templates
- Multi currency fund performance reporting
